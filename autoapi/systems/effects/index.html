systems.effects¶

class systems.effects.Effect(stat, value, mode='add', effect_type=EffectType.PERMANENT, source=None, duration=None, condition=None)[source]¶

Represents a stat modifier effect.

Parameters:

  • stat (str) – The name of the stat to modify (e.g., ‘strength’).

  • value (float) – The value to add (or multiply, if mode is ‘mul’).

  • mode (str, optional) – ‘add’ for additive, ‘mul’ for multiplicative (default ‘add’).

  • effect_type (EffectType) – The type/duration of the effect.

  • source (Any, optional) – The source of the effect (item, skill, etc.).

  • duration (int, optional) – Number of turns for temporary effects.

  • condition (Callable, optional) – Function to check if the effect is active (for conditioned/contextual).
